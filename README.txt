zadanie "Introsort - projekt"
==============================

To repo jest dostępne na Githubie:
    https://github.com/ArchieT/theintrosort

Aby budować, potrzebny CMake. W wersji co najmniej 3.9.
I musi być wersja GCC taka żeby C++11.

Aby zbudować maina:

```
cmake && make theintrosort
```



Aby zbudować unittesty:

`cmake && make tistest`
